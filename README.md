# 🍴 グルメファインダー - 使用方法

## 📋 概要

ホットペッパーグルメAPIを使用した、現在地周辺のレストランを検索できるアプリです。

## 🔑 APIキーの取得方法（無料・約10分）

### 1. リクルートWEBサービスにアクセス
https://webservice.recruit.co.jp/register

### 2. 新規登録
- メールアドレスを入力
- 届いたメールから登録を完了

### 3. APIキーの発行
- ログイン後、APIキーが自動発行されます
- このAPIキーをアプリに入力してください

**注意事項：**
- APIキーは無料で利用できます
- 商用利用は禁止されています（個人利用のみ）
- クレジット表記が必要です（アプリに既に含まれています）

## 🚀 アプリの使い方

### 1. APIキーの設定
- アプリを開くと、まずAPIキー入力フォームが表示されます
- 取得したAPIキーを入力して「設定」ボタンをクリック

### 2. 位置情報の許可
- ブラウザから位置情報の使用許可を求められたら「許可」をクリック
- 自動的に現在地周辺のお店が検索されます

### 3. フィルター機能
**検索範囲**
- 300m / 500m / 1000m / 2000m / 3000m から選択

**料理ジャンル**
- 居酒屋、中華、イタリアン、ラーメン、カフェなど18種類から選択
- 「すべて」を選択するとジャンル指定なしで検索

**予算**
- 〜500円から30,001円以上まで、14段階の予算帯から選択
- 「すべて」を選択すると予算指定なしで検索

### 4. お店の詳細を見る
- 各お店のカードをクリックすると、ホットペッパーグルメの詳細ページが開きます
- 予約や詳しいメニュー、口コミなどを確認できます

## ✨ 機能

- 📍 **位置情報ベースの検索**: 現在地から近い順に表示
- 🎯 **詳細フィルター**: ジャンル・予算・検索範囲で絞り込み
- ⭐ **おすすめ表示**: 検索結果の上位3件を「おすすめ」として表示
- 🔄 **リアルタイム更新**: フィルター変更時に自動で再検索
- 📱 **レスポンシブデザイン**: スマホ・タブレット・PCに対応

## 🛠️ 技術仕様

- **API**: ホットペッパーグルメ Webサービス
- **位置情報**: Geolocation API
- **最大取得件数**: 50店舗
- **更新**: リアルタイム（フィルター変更時に自動再検索）

## ⚠️ トラブルシューティング

### 「位置情報を取得できませんでした」と表示される
- ブラウザの位置情報設定を確認してください
- プライベートモード/シークレットモードでは動作しない場合があります

### 「API呼び出しに失敗しました」と表示される
- APIキーが正しく入力されているか確認してください
- インターネット接続を確認してください
- APIの利用制限に達していないか確認してください

### 検索結果が0件
- 検索範囲を広げてみてください
- フィルター条件を緩めてみてください
- その地域に該当するお店が登録されていない可能性があります
